docker-lang-r
=============

SYNOPSIS
--------

``` bash
  $ mkdir /opt/R
  $ docker pull kuniyoshi/lang-r
  $ docker run -i -t -v /opt/R/:/opt/R:rw kuniyoshi/lang-r
```

DESCRIPTION
-----------

This repository provides a `Dockerfile`, and some libraries.

The `Dockerfile` is used to build container image for the docker.
